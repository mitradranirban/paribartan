Paribartan Variable Color Typeface

পরিবর্তন রঙীন টাইপফেস

Paribartan is a WIP variable color font created using open source tools

It covers Latin, Extended Latin and Bangla (Bengali) characters

It has Five in-built color palettes

Orange , Red , Grey , Green , and Blue any mix in between

You can also do Color Animation using css.

It has 2 custom axes of Shadow and Mask . which is useful to do 3 D Effect using css.
